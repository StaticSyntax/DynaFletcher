## Features
• Cuts and strings any type of bow.  
• Cuts arrow shafts and javelin shafts.  
• Flights headless arrows, unfinished bolts and darts.  
• Cuts gems into bolt tips.  
• Tips arrows, bolts and javelins.  
• Cuts all wooden shields.
